# Cronómetros de Entrenamiento

Proyecto Vite + React + Supabase para manejar varios cronómetros deportivos con sonidos.

## Ejecutar en local

```bash
npm install
cp .env.example .env.local
npm run dev
```

Completa `.env.local` con tus datos de Supabase.

## Supabase

Ejecuta el archivo `supabase/schema.sql` en el SQL Editor de Supabase.
